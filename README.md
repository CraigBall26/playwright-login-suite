# **Playwright Login Suite**

A lightweight, maintainable automation suite built with **Playwright** and **pytest**, focused on validating login flows with clean patterns and fast execution. This project demonstrates practical automation approaches used in real products.

---

## **✨ Key Features**
- **Page‑Object‑style structure** for clarity and reuse  
- **Environment‑driven configuration** for flexible test execution  
- **Readable, structured test design**  
- **Fast execution** suitable for CI pipelines  
- **Clear reporting** and trace output  
- Includes **AI‑assisted refactoring** and test‑idea exploration  

---

## **🛠 Tech Used**
- **Python**  
- **Playwright**  
- **pytest**  

---

## **🎯 Why This Project Exists**
A focused automation project created to sharpen my Playwright skills, explore clean test architecture and maintain strong engineering habits while between roles. Also used as a sandbox for experimenting with **AI‑assisted development** and test‑design iteration.

---

## **📌 Notes**
This repo is intentionally lightweight — the goal is clarity, maintainability and demonstrating practical automation patterns rather than building an over‑engineered framework.
